Tag <h1> até <h6>

(Voltar)

As tags HTML que usamos para criar títulos em páginas web são: <h1>, <h2>, <h3>, <h4>, <h5> e <h6> . O significado dessas tags é o mesmo, que é definir títulos, mas o nível de importância é diferente para cada uma, onde <h1> tem o nível mais alto e <h6> o mais baixo.

Tag <p>

(Voltar)

A tag <p> é utilizada para demarcar um parágrafo. O elemento HTML <p> (ou Elemento HTML Parágrafo) representa um parágrafo do texto. Parágrafos são, geralmente, representados em mídia visual , como bloco de texto que são separados dos blocos adjacente por espaços brancos verticais e/ou recuo de primeira-linha.

Tag <mark>

(Voltar)

É usado para destacar a parte do texto em um parágrafo. A tag <mark> é nova no HTML 5.Por padrão, a tag <mark> destaca o conteúdo do texto na cor amarela.

Tag <small>

(Voltar)

O marcador <Small> Small Element ou fonte menor é utilizado para reduzir o tamanho da fonte num intervalo definido de um texto.

Tag <i>

(Voltar)

A tag <i> pode representar uma parte do texto com uma diferença semântica, na qual a representação tipográfica padrão é no tipo itálico. Isso significa que os navegadores continuarão a apresentar o conteúdo em itálico, mas, de acordo com o que é definido, isso não é mais requerido.

Tag <u>

(Voltar)

O marcador <U></U> "Underline Element" ou sublinhado é utilizado para sublinhar um espaço definido de um texto.

Tag <strong>

(Voltar)

O elemento <strong> é utilizado em conteúdos que são de "grande importância", incluindo coisas de urgentes (como alertas). Podem ser sentenças que são de grande importância para toda a página, ou, você pode meramente tentar pontuar que algumas palavras são de grande importância, comparado ao conteúdo próximo.

Tag <ol>

(Voltar)

O Elemento HTML <ol> (ou Elemento HTML de lista ordenada) representa uma lista de itens ordenados. De forma característica esses itens ordenados em uma lista são mostrados com uma contagem que os precede, que pode ser de qualquer tipo, como numerais, letras, algarismos romanos, ou simples símbolos.

Tag <ul>

(Voltar)

O elemento HTML <ul> (ou elemento HTML de Lista desordenada) representa uma lista de itens sem ordem rígida, isto é, uma coleção de itens que não trazem uma ordenação numérica e as suas posições, nessa lista, são irrelevantes.

Tag <a>

(Voltar)

A tag <a> é usada para criar uma âncora de onde vincular, o atributo href é usado para onde endereçar o documento, e as palavras entre as tags de abertura e fechamento da âncora serão exibidas como um hipervínculo.

Tag <hr>

(Voltar)

O elemento HTML <hr> representa uma quebra temática entre elementos de nível de parágrafo (por exemplo , uma mudança da cena de uma história, ou uma mudança de tema com uma seção). Nas versões anteriores do HTML, representava uma linha horizontal.

Tag <sub>

(Voltar)

A tag <sub> é usada para adicionar um texto subscrito ao documento HTML. A tag <sub> define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor.

Tag <sup>

(Voltar)

O elemento sup: Este é um elemento inline destinado a marcar fragmentos de texto que devam ser renderizados em alinhamento vertical mais alto em relação ao texto principal.

Tag <blockquote>

(Voltar)

O Elemento HTML <blockquote> (ou Elemento HTML de citação de bloco) indica que o texto incluído é uma longa citação. Normalmente, este é processado visualmente pelo recuo.
